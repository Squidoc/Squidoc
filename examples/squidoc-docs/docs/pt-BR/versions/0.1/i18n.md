# Internacionalização

Esta página se concentra em docs multilíngues e explica como essa parte funciona em um projeto Squidoc.

## O que você vai configurar

Você vai aprender a manter locales, rotas traduzidas e rótulos de navegação.

## O que conferir antes de publicar

Atualize todos os idiomas quando o inglês ganhar páginas ou exemplos.

## Leia também

[Configuração](/configuration) · [Plugins](/plugins) · [Deploy](/deployment)

## Exemplos

```ts
plugins: [
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```

```txt
docs/
  index.md
  configuration.md
```

```txt
docs/
  es/
    index.md
    configuration.md
```

```ts
plugins: [
  {
    name: "@squidoc/plugin-versions",
    options: {
      versions: [{ name: "0.1", label: "0.1" }],
    },
  },
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```

```txt
docs/
  versions/
    0.1/
      configuration.md
  es/
    versions/
      0.1/
        configuration.md
```
