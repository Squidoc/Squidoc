# Squidoc

Squidoc is a static-first documentation platform with a plugin system, powerful themes, and SEO-first defaults.

The project is currently in phase one. The first goal is to generate a project, render Markdown and MDX docs, configure plugins and themes, and ship a static site that is easy to deploy.

Start with [Getting Started](/getting-started), then read about [configuration](/configuration) and the [architecture](/architecture).
