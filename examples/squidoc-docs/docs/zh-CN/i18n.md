# 国际化

本页聚焦 多语言文档，说明它在 Squidoc 项目中的工作方式。

## 你将配置什么

你将学习如何维护 locale、翻译路由和本地化导航标签。

## 发布前检查

英文文档新增页面或示例时，请更新所有支持语言。

## 示例

```ts
plugins: [
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```
