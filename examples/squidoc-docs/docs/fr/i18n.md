# Internationalisation

Cette page se concentre sur des docs multilingues et explique comment cette partie fonctionne dans un projet Squidoc.

## Ce que vous allez configurer

Vous allez apprendre à gérer locales, routes traduites et libellés de navigation.

## À vérifier avant publication

Mettez à jour toutes les langues quand l’anglais reçoit pages ou exemples.

## À lire aussi

[Configuration](/configuration) · [Plugins](/plugins) · [Déploiement](/deployment)

## Exemples

```ts
plugins: [
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```

```txt
docs/
  index.md
  configuration.md
```

```txt
docs/
  es/
    index.md
    configuration.md
```

```ts
plugins: [
  {
    name: "@squidoc/plugin-versions",
    options: {
      versions: [{ name: "0.1", label: "0.1" }],
    },
  },
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```

```txt
docs/
  versions/
    0.1/
      configuration.md
  es/
    versions/
      0.1/

```
