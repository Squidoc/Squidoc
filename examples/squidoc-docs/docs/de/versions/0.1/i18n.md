# Internationalisierung

Diese Seite konzentriert sich auf mehrsprachige Docs und erklärt, wie dieser Teil in einem Squidoc-Projekt funktioniert.

## Was du einstellst

Hier lernst du, wie du Locales, übersetzte Routen und lokalisierte Navigationslabels pflegen kannst.

## Vor dem Veröffentlichen prüfen

Aktualisiere alle unterstützten Sprachen, wenn englische Docs neue Seiten oder Beispiele bekommen.

## Weitere Themen

[Konfiguration](/configuration) · [Plugins](/plugins) · [Deployment](/deployment)

## Beispiele

```ts
plugins: [
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```

```txt
docs/
  index.md
  configuration.md
```

```txt
docs/
  es/
    index.md
    configuration.md
```

```ts
plugins: [
  {
```
