# Internacionalización

Squidoc 0.1 incluye la base inicial de i18n: rutas con prefijo de idioma, selector de idioma, búsqueda filtrada y etiquetas SEO alternas.

## Rutas

El idioma principal se mantiene en `/docs`. Los idiomas adicionales usan `/{locale}/docs`.
