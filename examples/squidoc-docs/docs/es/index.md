# Introducción

Squidoc convierte Markdown en documentación estática rápida, lista para SEO y fácil de extender con plugins y temas.

Esta página existe para probar el soporte de internacionalización antes del lanzamiento. El contenido completo en español puede crecer con el proyecto, pero las rutas, el selector de idioma, la búsqueda y las etiquetas SEO ya usan la misma estructura que tendrá el sitio real.

## Empieza aquí

La documentación principal en español vive bajo `/es/docs`. Si una página todavía no tiene traducción, el selector de idioma vuelve al inicio de la documentación para ese idioma.
