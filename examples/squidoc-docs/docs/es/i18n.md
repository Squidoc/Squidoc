# Internacionalización

El plugin i18n publica la misma documentación en varios idiomas y mantiene rutas por locale.

## Qué vas a configurar

Declara los locales soportados, traduce los documentos bajo cada carpeta de idioma y localiza labels de navegación cuando uses navegación manual.

## Qué revisar antes de publicar

Mantén los idiomas en sincronía con el inglés; si agregas una página o un ejemplo, actualiza todas las versiones localizadas.

## Ejemplos

```ts
plugins: [
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```

```txt
docs/
  index.md
  configuration.md
```

```txt
docs/
  es/
    index.md
    configuration.md
```

```ts
plugins: [
  {
    name: "@squidoc/plugin-versions",
    options: {

    name: "@squidoc/plugin-i18n",
```
