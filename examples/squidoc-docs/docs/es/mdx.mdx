# MDX

El plugin MDX permite descubrir archivos `.mdx` junto a tus documentos Markdown.

## Qué vas a configurar

Úsalo cuando quieras preparar contenido MDX dentro del árbol de docs; la compatibilidad con componentes JSX completos todavía es limitada.

## Qué revisar antes de publicar

Verifica el build cada vez que agregues MDX para confirmar que el pipeline de documentos reconoce el archivo como esperas.

## También puedes leer

[Configuración](/configuration) · [Plugins](/plugins) · [Deployment](/deployment)

## Ejemplos

```bash
npx squidoc add plugin @squidoc/plugin-mdx
```

```ts
export default defineConfig({
  plugins: [
    "@squidoc/plugin-seo",
    "@squidoc/plugin-codeblocks",
    "@squidoc/plugin-article-tree",
    "@squidoc/plugin-mdx",
  ],
});
```

```mdx
---
title: API Guide
description: A guide written in an MDX file.
---

# API Guide

Regular Markdown content works here.
```

```mdx
import Callout from "../components/Callout";

<Callout>This is not supported yet.</Callout>
```
