# 国際化

このページでは 多言語ドキュメント に焦点を当て、Squidoc プロジェクト内でどのように機能するかを説明します。

## 設定する内容

ここでは ロケール、翻訳済みルート、ナビゲーションラベルを管理します。

## 公開前に確認すること

英語版にページや例を追加したら、対応する全言語を更新してください。

## 例

```ts
plugins: [
  {
    name: "@squidoc/plugin-i18n",
    options: {
      defaultLocale: "en",
      locales: [
        { code: "en", label: "English" },
        { code: "es", label: "Español" },
      ],
    },
  },
];
```
